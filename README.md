# 🇧🇷 Seleção Brasileira — Prancheta Tática

Projeto de análise tática interativa da Seleção Brasileira, desenvolvido com HTML, CSS e JavaScript puro.

## 🔥 Funcionalidades

- ⚽ Campo de futebol interativo com arrastar e soltar
- 🃏 Cards FIFA com atributos animados
- 📋 Troca de formações com animação suave
- 🔗 Linhas de entrosamento entre jogadores
- 🔥 Mapa de calor por posição
- ✏️ Ferramentas táticas — setas, curvas, círculos, zonas e texto
- 🔄 Sistema de substituições com apito
- 📊 Análise tática inteligente com penalização por posição
- 🎙️ Narração com 4 estilos diferentes
- 📸 Exportar a prancheta como imagem PNG
- 🎬 Tela de entrada com vídeo e tutorial interativo
- 🎨 5 temas visuais do campo

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript Vanilla
- Canvas API
- Web Audio API
- html2canvas
